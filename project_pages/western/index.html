Using snow cover extracted from satellite imagery, NLCD land cover, and elevation data, I built a terrain map of the legendary Western States 100 race course. I rotated the map(and the hillshade's light source) about 65 degrees to enable vertical scrolling from start to finish.

Now I just need to qualify for the race so I can ground truth the map.

Tools: Blender, Photoshop, Illustrator, QGIS
